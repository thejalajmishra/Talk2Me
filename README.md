# Talk2Me

Talk2Me is an AI Communication & Accent Coaching Tool.

## Tech Stack
*   **Frontend**: React (Vite), Tailwind CSS, Lucide React
*   **Backend**: Python (FastAPI), SQLAlchemy, Librosa, OpenAI (Mocked for demo)
*   **Database**: SQLite (Dev) / PostgreSQL (Prod)

## Setup

### Backend
1.  `cd backend`
2.  `./run.sh` (or `../.venv/bin/uvicorn main:app --reload`)

### Frontend
1.  `cd frontend`
2.  `npm run dev`

## Features
*   Topic Library
*   Audio Recording & Analysis
*   WPM & Filler Word Detection
*   Leaderboard
